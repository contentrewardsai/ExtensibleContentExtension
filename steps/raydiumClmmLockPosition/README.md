# Raydium CLMM lock position

**`CFS_RAYDIUM_CLMM_LOCK_POSITION`** — `raydium.clmm.lockPosition` for an **unlocked** position the automation wallet owns. Produces a **lock** NFT; use **raydiumClmmHarvestLockPosition** with that lock mint to claim lock rewards.
