# bscWatchRefresh

Sends **`CFS_BSC_WATCH_REFRESH_NOW`** to the service worker (`background/bsc-watch.js`) so workflows can force a BscScan poll between steps.

Configure a **BscScan API key** under **Settings → BSC / PancakeSwap**. Watched addresses come from **Following** wallets (`evm`, BSC or Chapel, Watch on).

See **docs/BSC_AUTOMATION.md** for the full BSC automation picture.
