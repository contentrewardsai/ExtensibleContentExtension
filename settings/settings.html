ShotStack Keys

API keys for ShotStack cloud video/audio rendering. Get keys from dashboard.shotstack.io.

Upload Post Profiles

Profiles from your local API key and remote backend, deduplicated.

JWT Token Management

JWT tokens for Upload Post profiles expire every 48 hours. They are automatically refreshed daily.

Upload Post Platform Defaults

Set default values for each platform. Per-profile defaults override global defaults. Priority (highest to lowest): generator fields, workflow variables, profile+platform defaults, global platform defaults.
