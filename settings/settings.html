  1. Click the button above — a file picker opens in mcp-server/dist/
  2. Find the binary for your OS: StartMacMCPServer (Apple Silicon), StartMacIntelMCPServer (Intel Mac), StartWindowsMCPServer.exe, or StartLinuxMCPServer
  3. Right-click → Show in Finder (or Explorer), then double-click to run
