# Local AI models

The repo tracks this folder (see `.gitignore`: `models/*` with exceptions for `README.md` and `.gitkeep`). Large weights under `models/Xenova/` stay untracked. LaMini is saved into **your project folder** when you set it in the side panel (and **Download LaMini** if needed), under `models/Xenova/LaMini-Flan-T5-783M/`.

Optional CLI (same layout):

- `./scripts/download-lamini-model.sh` — Xenova/LaMini-Flan-T5-783M (~820MB), used by the LLM step and Local AI Chat
